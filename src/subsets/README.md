[Subsets](https://shivajivarma.com/code-base/2018/05/10/subsets/)
===============

> Program to print all subsets of given set of letters

Input:

arrSize	// array size.

arr[ ] 	// array.

Output: subsets of letters.

----
### Example
input: 
```
ENTER THE SIZE OF THE ARRAY:
4

ENTER THE ELEMENTS INTO THE ARRAY:
a
b
c
d
```

output:	

```
All combinations:
{ }
{ d }
{ c }
{ c d }
{ b }
{ b d }
{ b c }
{ b c d }
{ a }
{ a d }
{ a c }
{ a c d }
{ a b }
{ a b d }
{ a b c }
{ a b c d }
```
